# enzimath

Este repositorio contiene los Jupyter Notebooks de las 4 sesiones de *Enzimath: Entendiendo a las enzimas entre genes y ecuaciones*. Da click en alguna de las siguiente ligas para comenzar a usar el notebook correspondiente (técnicamente estarás creando una maquina virtual en un servidor en alguna parte del mundo). Binder permite crear esa maquina virtual usando la información contenida en este repositorio de GitHub.
